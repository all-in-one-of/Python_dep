In this scene (A_SCENE) all GUI functionality is enabled. But most will not works, especially for guests. This is mostly because this server does not have Afanasy, Python and your machine does not have this filesystem.

You can see cmd buttons with yellow back. This buttons are to run commands. Commands will be executed locally by your browser if cgru-addon installed.

  • Open - open folder in file explorer.
  • Nuke - open sequence in Nuke.
  • NkCp - copy sequence as Nuke Reader.
  • PdP - open sequence in PdPlayer (MSWin).
  • Pd++ - add sequence in opened PdPlayer (MSWin).
  • DJV - add sequence in DJView (Linux, MacOSX).
  • QT - open movie in QuickTime player (MSWin, MacOSX).
  • VLC - open movie in VLC player.
  • Totem - open movie in Totem player (Linux).
Command buttons can be customized and limited to specific platform(s).

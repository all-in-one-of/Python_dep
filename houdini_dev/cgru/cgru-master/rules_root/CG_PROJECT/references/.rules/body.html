You can generate thumbnails for almost all known image formats and folders that have images:

  • Thumbs: - Generate thumbnails.
  • [c] - Crop thumbnails to the same ascect.
  • - - Display thumbnails smaller.
  • + - Display thumbnails larger.

Unlike SHOT thumbnail, common image file thumbnail will not be generated and updated automatically.
